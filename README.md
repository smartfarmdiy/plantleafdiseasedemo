ฟีเจอร์ทำงานดังนี้:

Context-Aware: AI จะรับรู้ผลการวิเคราะห์ปัจจุบัน (เช่น ค่า N, P, K ที่ตรวจพบ) โดยอัตโนมัติ

Chat Interface: เกษตรกรสามารถพิมพ์ถามคำถามเพิ่มเติมได้ เช่น "มียี่ห้อปุ๋ยแนะนำไหม?" หรือ "ต้องรดน้ำยังไง?"

Generative Answers: ใช้พลังของ Gemini ในการตอบคำถามทางการเกษตรที่ซับซ้อนได้อย่างเป็นธรรมชาติและแม่นยำ

ฟังก์ชัน Score-CAM Heatmap (Class Activation Mapping) เพื่อจำลองการทำงานของ AI ในการ "เพ่งเล็ง" จุดสำคัญบนภาพใบพืช โดยจะแสดงผลในรูปแบบ Heatmap ไล่สี (น้ำเงิน --> เขียว --> เหลือง --> แดง) เพื่อบอกระดับความรุนแรงหรือความมั่นใจของ AI

สิ่งที่เพิ่มเติม:
AI Heatmap Layer: เลเยอร์แสดงแผนที่ความร้อน (Heatmap) ซ้อนทับบนใบพืช

Toggle Switch: ปุ่มเปิด-ปิด Heatmap แยกต่างหากจาก Segmentation

Visualization: จำลองสีแบบ Jet Colormap (เย็นไปร้อน) ตามหลักการ Score-CAM

#ประมวลผลภาพจริง" (Image Processing)
#(Real-time Color Detection):

Pixel Analysis: ระบบจะสแกนสีของทุกพิกเซลในภาพที่ถ่าย/อัปโหลด

Color Filtering:

สีเขียว (Green): มองว่าเป็นเนื้อเยื่อพืชปกติ -> ไม่วาด Mask

สีเหลือง/น้ำตาล/แดง (Yellow/Brown/Red): มองว่าเป็นรอยโรค (Chlorosis/Necrosis) -> วาด Mask สีแดงทับ

สีขาว/ดำ/เทา/ฟ้า: มองว่าเป็นพื้นหลัง -> ข้ามไป

ความแม่นยำ: Mask จะเกิดขึ้นตรงตำแหน่งที่เป็นสีเหลืองหรือน้ำตาลบนใบพืชจริงๆ เท่านั้น ทำให้ตรงกับความเป็นจริงมากที่สุด
