# plantleafdiseasedemo

#แอพตรวจสอบโรคใบพืชAI อย่างง่ายไว้ใช้อ้างอิง ทางเราได้ทำชุดข้อมูลไว้ใช้สำหรับเป็นข้อมูลอ้างอิงการใช้งานตรวจสอบโรคพืช
#สำหรับใครที่ต้องการนำไปพัฒนาต่อยอดหรือเอาไปเพิ่มความสามารถ

#app นี้เชื่อมต่อกับ api google gemini pro 3 

ฟีเจอร์ทำงานดังนี้:

Context-Aware: AI จะรับรู้ผลการวิเคราะห์ปัจจุบัน (เช่น ค่า N, P, K ที่ตรวจพบ) โดยอัตโนมัติ

Chat Interface: เกษตรกรสามารถพิมพ์ถามคำถามเพิ่มเติมได้ เช่น "มียี่ห้อปุ๋ยแนะนำไหม?" หรือ "ต้องรดน้ำยังไง?"

Generative Answers: ใช้พลังของ Gemini ในการตอบคำถามทางการเกษตรที่ซับซ้อนได้อย่างเป็นธรรมชาติและแม่นยำ
